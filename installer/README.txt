FBA费用计算器使用说明
=====================

功能介绍:
1. FBA配送费计算 - 输入商品尺寸和重量，计算亚马逊FBA配送费用
2. 重量转换工具 - 在不同重量单位之间进行转换
3. 货币转换工具 - 支持多种货币之间的汇率转换
4. 主题切换功能 - 支持多种颜色主题，个性化您的使用体验

安装指南:
1. 直接运行本文件夹中的"FBA费用计算器.exe"程序
2. 如需在桌面创建快捷方式，请右键点击可执行文件，选择"发送到" -> "桌面快捷方式"

使用方法:
1. 启动程序后，在主界面输入商品的长、宽、高(厘米)和重量(克)
2. 点击"计算配送费"按钮查看结果
3. 使用顶部导航栏切换到其他功能模块
4. 通过菜单栏可以切换不同的界面主题

注意事项:
- 本程序已优化UI界面，提供更直观、立体的操作体验
- 确保输入的尺寸和重量数据准确，以获得精确的计算结果
- 程序会自动保存您的主题偏好设置
- 如有任何问题，请关闭程序后重新启动

祝您使用愉快！
