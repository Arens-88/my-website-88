# FBA费用计算器

![GitHub release (latest by date)](https://img.shields.io/github/v/release/arens-88/fba-calculator)
![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-active-brightgreen)

专业的Amazon FBA费用计算工具，帮助卖家精确计算和分析FBA费用，优化产品定价策略。

## 项目介绍

FBA费用计算器是一款专为Amazon卖家设计的费用计算工具，能够根据最新的Amazon费率表，精确计算仓储费、配送费、佣金等各项费用，帮助卖家做出更明智的产品决策。

## 主要功能

- ✅ **精确费用计算**：根据最新Amazon费率表计算各项FBA费用
- ✅ **多维度分析**：提供仓储费、配送费、佣金等多维度费用分析
- ✅ **数据导出**：支持将计算结果导出为Excel或CSV格式
- ✅ **批量处理**：支持批量导入产品信息进行费用计算
- ✅ **仓储费用预估**：基于产品尺寸和重量预估长期仓储费用
- ✅ **多站点支持**：支持美国、欧洲、日本等多个Amazon站点

## 使用说明

1. 下载并安装最新版本的FBA费用计算器
2. 启动软件后，输入产品尺寸、重量、售价等信息
3. 点击"计算费用"按钮获取详细的FBA费用分析
4. 可选择导出计算结果或保存为项目文件
